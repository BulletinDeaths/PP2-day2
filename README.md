## 📖 О проекте
`onlineschool` — это учебный проект, представляющий собой веб-приложение, которое моделирует базовую функциональность систем управления образовательным процессом. Приложение позволяет управлять информацией об учащихся, преподавателях и курсах. Ключевой особенностью является возможность просмотра отчёта успеваемости студента. Данные проекта хранятся в простом файле data.json, а для взаимодействия с пользователем реализован простой веб-интерфейс.

---
## 📂 Структура проекта
```
/onlineschool
|-- app.py                 # Точка входа в приложение (основной Flask-скрипт)
|-- config.py              # Файл конфигурации приложения
|
|-- /models                # Модели данных и логика работы с БД
|   |-- __init__.py
|   |-- student.py          # Модель "Студент"
|   |-- teacher.py         # Модель "Преподаватель"
|   |-- course.py          # Модель "Курс"
|   |-- repository.py      # Логика чтения/записи данных в data.json
|
|-- /templates             # HTML-шаблоны для веб-интерфейса
|   |-- base.html          # Базовый шаблон (наследуется другими)
|   |-- index.html         # Главная страница
|   |-- students.html      # Список студентов
|   |-- teachers.html      # Список преподавателей
|   |-- courses.html       # Список курсов
|   |-- student_report.html# Отчёт успеваемости студента
|
└── data.json              # Файл с данными (заменяет базу данных)
```

## ⚙️ Примеры работы
**Вывод списка студентов:**
При переходе на страницу /students в браузере отображается список всех учащихся, зарегистрированных в системе.

**Вывод отчёта успеваемости:**
При переходе на страницу /student_report (или по ссылке из профиля студента) отображается подробный отчёт с оценками по всем курсам.

**Пример содержимого отчёта:**
- Студент: Иванов Иван
- Курс: Введение в программирование (Преподаватель: Петров П.П.)
- Оценки: 5, 4, 5, 5
- Средний балл: 4.75

## 💡 Использование

Шаг 1: Установка зависимостей
Установите необходимые библиотеки из файла requirements.txt.

```bash
pip install -r requirements.txt
```
*(Предполагается, что в файле указаны Flask и другие необходимые пакеты)*

Или установите вручную:

```bash
pip install flask
```

**Шаг 2: Запуск программы**
Запустите веб-приложение из терминала в корневой папке проекта.

```bash
python app.py
```
По умолчанию приложение запустится на http://127.0.0.1:5000. Откройте этот адрес в вашем браузере.

## 🧠 Логика анализа
Алгоритм работы приложения:

**1. Инициализация:** При запуске app.py происходит инициализация Flask-приложения и загрузка конфигураций из config.py.

**2. Загрузка данных:** При первом запросе к данным (например, для отображения списка студентов) модуль repository.py считывает информацию из файла data.json.

**3. Обработка запроса:** Flask-маршруты (routes) в app.py определяют, какую страницу запросил пользователь.

**4. Взаимодействие с моделями:** Для выполнения операций (например, создание нового курса) вызываются методы из классов моделей (Course, Student, Teacher).

**5. Сохранение данных:** После любых изменений (добавление студента, выставление оценки) обновлённые данные сериализуются и записываются обратно в файл data.json через repository.py.

**6. Отображение:** Данные передаются в соответствующий HTML-шаблон из папки /templates, который генерирует финальную веб-страницу.

**7. Генерация отчёта:** Для создания отчёта успеваемости логика собирает все оценки конкретного студента из структуры данных, вычисляет средний балл и форматирует результат для вывода на странице.
